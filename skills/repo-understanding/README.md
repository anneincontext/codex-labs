# Repo Understanding

**English** | [中文](README_zh.md)

Map an unfamiliar codebase: layout, entry points, build/test commands, conventions.

## Outputs

- High-level map (folders, binaries, key crates/modules)
- “Start here” reading order
- How to build, test, and run locally

## Human checks

- Entry points match what you see when you open the repo
- Commands actually work on your machine
