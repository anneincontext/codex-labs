# Repo Understanding

[English](README.md) | **中文**

摸清陌生代码库：结构、入口、构建/测试命令、约定。

## 产出

- 高层地图（目录、二进制、关键 crate/模块）
- 「从这里读起」的顺序
- 本地如何 build、test、run

## 人工核对

- 入口是否与你打开仓库后看到的一致
- 命令在你机器上是否真能跑通
