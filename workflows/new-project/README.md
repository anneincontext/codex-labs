# New Project

**English** | [中文](README_zh.md)

Onboard to a codebase you have not worked in before.

## Skill sequence

```text
repo-understanding → architecture → documentation → testing
```

| Step | Skill | Goal |
| ---- | ----- | ---- |
| 1 | [repo-understanding](../../skills/repo-understanding/) | Map layout, build, test, conventions |
| 2 | [architecture](../../skills/architecture/) | One-page system mental model |
| 3 | [documentation](../../skills/documentation/) | Personal onboarding note in the lab |
| 4 | [testing](../../skills/testing/) | Run baseline tests; note gaps |

## Human gates

- After step 1: you can run the project locally
- After step 4: baseline test result recorded in a [case](../../cases/)

## Done when

You can explain the repo to someone else in five minutes and know where to start coding.
