# Release

[English](README.md) | **中文**

为交付做准备：质量线、文档、最终 review。

## 技能序列

```text
testing → documentation → code-review
```

| 步骤 | Skill | 目标 |
| ---- | ----- | ---- |
| 1 | [testing](../../skills/testing/) | 相关测试全跑；无已知 flake |
| 2 | [documentation](../../skills/documentation/) | Changelog、README、必要时迁移说明 |
| 3 | [code-review](../../skills/code-review/) | 对发布 diff 最后一遍 |

## 人工关卡

- 版本号与 breaking change：你拍板
- 生产发布：不要单独交给 Codex

## 完成标准

你今天愿意对用户发布，且有清晰的回滚方案。
